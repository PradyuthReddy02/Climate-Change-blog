This project focuses on collecting climate data of india using satellite images and data 
to create a blog on the climate change the country is experiencing

the following datasets have been used to collect the following features:
flooding:https://developers.google.com/earth-engine/datasets/catalog/JRC_GSW1_4_MonthlyHistory#bands
temperature:https://developers.google.com/earth-engine/datasets/catalog/NOAA_CPC_Temperature?hl=en#bands
precipitation:https://developers.google.com/earth-engine/datasets/catalog/UCSB-CHG_CHIRPS_DAILY#description
tree cover:https://developers.google.com/earth-engine/datasets/catalog/MODIS_006_MOD44B

most the scripts require a google earth engine account and the permissions
signup here:https://earthengine.google.com/signup/

To access the blog, run the blog.html script in the blog folder
The main.ipynb contains most of the scripts used for data collection through the apis
the visualization.ipynb is a secondary script which was used to visualize the data and generate interactive maps
